# YAML Configuration Expertise

## YAML Fundamentals

### Syntax Rules
- Indentation-based structure (spaces, not tabs)
- Key-value pairs separated by colons
- Lists with dash notation
- Comments with hash symbols
- String quoting rules
- Multi-line strings

### Data Types
- Strings (quoted and unquoted)
- Numbers (integers and floats)
- Booleans (true/false)
- Nulls (null or ~)
- Arrays and sequences
- Nested objects
